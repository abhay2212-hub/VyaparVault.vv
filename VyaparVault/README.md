# VYAPARVAULT - India's Next Major Hygiene Brand

VYAPARVAULT is a full-stack, production-ready eCommerce website for bathroom and toilet cleaners. Designed for a premium, national-level FMCG feel, it's 100% responsive and feature-rich.

## Features
- **Frontend**: Next.js App Router, Tailwind CSS, Framer Motion
- **Backend**: Node.js + Express, MongoDB, JWT
- **Payments**: Razorpay Integration, Cash on Delivery
- **Responsive Design**: Optimized for Mobile, Desktop, and Tablet
- **Animations**: High-impact scroll reveals and interactive elements
- **Admin**: Full-featured administrative dashboard for sales & inventory

## Quick Start (Local Environment)

### Prerequisites
- Node.js (v18+)
- MongoDB (Running locally or on Atlas)
- Razorpay API Keys

### Step 1: Backend Setup
1. `cd backend`
2. `npm install`
3. Edit `.env` with your MongoDB URI and Razorpay keys
4. `node seeder.js` (Optional: Seeds the database with sample products)
5. `npm start` (Runs on port 5000)

### Step 2: Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev` (Runs on port 3000)
